# WhatsApp Funding Application Bot

A complete WhatsApp bot for collecting funding applications that saves directly to your Airtable.

## Features

- ✅ Complete application flow matching your Airtable schema
- ✅ South African ID validation
- ✅ Save and continue functionality
- ✅ Progress tracking
- ✅ Data validation
- ✅ Airtable integration
- ✅ Meta WhatsApp Business API
- ✅ Session management
- ✅ Error handling and logging
